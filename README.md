# 🥜 심봤다 (Jaego) - 프론트엔드

동네 남는 재고, 도토리를 콕! 우리 동네 숨은 도토리를 찾는 재미있는 플랫폼입니다.

## ✨ 주요 기능

- 🗺️ **Google Maps 기반** 상품 탐색
- 📍 **현재 위치 기반** 상품 추천
- 🔍 **카테고리별 필터링** 및 검색
- 💝 **찜 기능** 및 상품 관리
- 🛒 **실시간 예약 시스템**
- 📱 **모바일 우선** 반응형 디자인

## 🏗️ 기술 스택

- **React 18** + **Vite**
- **Styled Components** + **CSS3**
- **Google Maps JavaScript API**
- **JWT Token** 
- **Netlify** 

## 📁 프로젝트 구조

```
src/
├── api/          # API 통신 모듈
├── components/   # 재사용 컴포넌트
├── contexts/     # React Context
├── hooks/        # 커스텀 훅
├── pages/        # 페이지 컴포넌트
└── styles/       # 전역 스타일
```

## 🔧 주요 컴포넌트

- **AdvancedGoogleMap**: Google Maps 통합, 핀 클러스터링
- **SearchBar**: 실시간 검색 및 필터링
- **BottomSheet**: 상품 목록 표시
- **ProductCard**: 상품 정보 카드

## 🌐 API 연동

- **Base URL**: `https://yeonhee.shop`
- **인증**: JWT Bearer Token
- **주요 API**: 상품 조회, 추천, 예약, 알림

- 터치 제스처 지원
- PWA 준비
